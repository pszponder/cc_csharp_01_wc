# Word Count

This is a solution to [John Crickett's Coding Challenge #1 - Build your own wc!](https://codingchallenges.substack.com/p/coding-challenge-1)

## Usage

```bash
# Build the Project
make build

# Run the Project
make run

# Clean the Project
make clean
```
